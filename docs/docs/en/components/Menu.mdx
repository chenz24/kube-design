## Usage

The menu provides categories and functions, and uses a multi-level structure to organize and arrange the site structure.

## Basic Usage

Nest the MenuItem component in the Menu component.

Using disabled can disable the MenuItem component.

Using OnClick method can bind events to MenuItem component.

Using as can modify the component type, such as `a` label, you can add jump link to the MenuItem.

```js live=true
() => {
  const { Add, Stop, Pen, Trash } = KubeIcon;
  const clickCallback = () => {
     console.log('you click me')
  }
  return (
    <Card style={{ width: '400px' }}>
      <Menu width={300}>
        <MenuLabel>menu label</MenuLabel>
        <MenuItem icon={<Add />} onClick={clickCallback}>创建</MenuItem>
        <MenuItem icon={<Stop />} disabled>停止调度</MenuItem>
        <MenuItem icon={<Pen />} as="a" href="/login">Edit</MenuItem>
        <Divider />
        <MenuItem icon={<Trash />}>Delete</MenuItem>
      </Menu>
    </Card>
  )
}
```
