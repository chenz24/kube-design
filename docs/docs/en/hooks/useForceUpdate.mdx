## Usage

Hook returns a function, when this function is called component rerenders:

```js live=true
() => {
  const forceUpdate = useForceUpdate();
  return (
     <Group position="center">
       <Text>{randomId()}</Text>
       <Button onClick={forceUpdate}>Force update</Button>
     </Group>
  );
}
```

## Definition

```tsx
function useForceUpdate(): () => void;
```
