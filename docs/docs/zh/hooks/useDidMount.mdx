## 使用方法

这个函数在组件被挂载后执行。useEffect在不传入依赖时会在组件第一次挂载时执行这个函数。

```js
useEffect(() => {
  if (typeof callback === 'function') {
     fn();
  }
}, []);
```

useDidMount 使用相同的逻辑:

```js
useDidMount(() => console.log('component is mounted'));
```

## 定义

```js
function useDidMount(fn:() => void) : void;
```
