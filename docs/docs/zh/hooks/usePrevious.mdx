## Usage

`usePrevious` hook can be used to store props or previous state

```js live=true
() => {
  const [value, setValue] = React.useState(0);

  const lastValue = usePrevious(value);
  return (
    <div>
      <p>Current: {value} - previous:{lastValue}</p>
      <Button onClick={()=>setValue(value+1)}>Increment number</Button>
    </div>
  )
}
```
